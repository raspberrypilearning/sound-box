## Challenge

--- task ---

Make one more room for the Bug Hotel.

Your new room should have:

- one clear bug
- one simple thing to fix
- one reward that helps the player move on

You could try:

- a class name typo that breaks the layout
- a hidden message that should change from `display: none` to visible
- a sound button that uses the wrong file name
- a hover effect that hides the clue instead of showing it

Give your room its own file name and link it from `checkout.html`.

--- /task ---
